# Lab 6.1 — สถาปัตยกรรม Dashboard (3 ชั้น)

## แบบฝึก: วาด Wireframe บนกระดาษ A4

อ้างอิง wireframe ในเอกสารหน่วยที่ 6 กำหนด:

1. หัวข้อ Dashboard
2. KPI Card อย่างน้อย 3 ตัว
3. กราฟอย่างน้อย 3 ประเภท (Bar / Line / Pie หรือ Doughnut)
4. ตาราง — คอลัมน์ที่จะแสดง

## Wireframe (อ้างอิง Lab จริง)

```
[KPI x4]
[ Bar: Top 10 จังหวัด ] [ Pie: ยานพาหนะ ]
[ Doughnut: ความรุนแรง ] [ Line: รายเดือน ]   ← คู่กัน ไม่เต็มหน้า
[ ตาราง + ค้นหา + Export CSV ]
```
