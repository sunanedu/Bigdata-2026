Lab 1.3 — ERD ร้านป้าแดง (Workshop 1.3)
========================================

ความสัมพันธ์:
  customers 1 ──< orders 1 ──< order_items >── 1 products

┌────────────┐     ┌────────────┐     ┌────────────┐
│  customers │     │   orders   │     │  products  │
│  (ลูกค้า)  │     │ (ใบสั่งซื้อ) │    │  (สินค้า)  │
├────────────┤     ├────────────┤     ├────────────┤
│cust_id  PK │─┐   │order_id PK │   ┌─│product_idPK│
│cust_name   │ │   │cust_id  FK │◄──┘ │prod_name   │
│phone       │ └──►│order_date  │     │price       │
│address     │     │total_price │     │category    │
└────────────┘     └────────────┘     │stock_qty   │
                        │             └────────────┘
                        │ 1:N
                        ▼
                ┌────────────────┐
                │  order_items   │
                │ (รายการสินค้า) │
                ├────────────────┤
                │item_id     PK  │
                │order_id    FK  │
                │product_id  FK  │
                │quantity        │
                │unit_price      │
                └────────────────┘

ประเภทความสัมพันธ์:
  - customers → orders     : 1:N (ลูกค้า 1 คน สั่งซื้อได้หลายครั้ง)
  - orders → order_items   : 1:N (ใบสั่งซื้อ 1 ใบ มีหลายรายการ)
  - products → order_items : 1:N (สินค้า 1 ชนิด อยู่ในหลายใบสั่งซื้อ)
  - products ↔ customers   : M:N ผ่านตาราง order_items (Junction Table)
