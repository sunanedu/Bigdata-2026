## คำสั่ง

เลือก **หนึ่งแหล่งข้อมูล** ที่คุณใช้ในชีวิตจริง (เช่น TikTok, โรงเรียน, ร้านสะดวกซื้อ, โรงพยาบาล) แล้วตอบตารางด้านล่าง

| 5V | คำถามช่วยคิด | ตัวอย่างของคุณ |
|----|-------------|----------------|
| **Volume** | ข้อมูลมีมากแค่ไหน? เก็บกี่ GB/TB? | |
| **Velocity** | ข้อมูลเกิดเร็วแค่ไหน? ต้องประมวลผลทันทีไหม? | |
| **Variety** | มีกี่รูปแบบ? (ตาราง, รูป, วิดีโอ, JSON) | |
| **Veracity** | ข้อมูลอาจผิดหรือไม่น่าเชื่อถือตรงไหน? | |
| **Value** | นำไปตัดสินใจอะไรได้จริง? | |

## คำถามเพิ่มเติม

1. แหล่งข้อมูลที่เลือก ใช้ **Excel** จัดการได้สบายหรือไม่? ทำไม?
2. วาด **Data Lifecycle** 5 ขั้น (เก็บ → ประมวลผล → วิเคราะห์ → นำเสนอ → ตัดสินใจ) สำหรับแหล่งข้อมูลนั้น
